# **Epic Battle Music**

Turn every boss fight into a cinematic moment.

Fed up with battling powerful bosses in silence? **Epic Battle Music** injects drama into your Minecraft adventures by triggering custom boss themes the moment the fight begins.

Whether you're using boss mods or sticking to vanilla, any creature with over **200 HP** will activate an intense soundtrack that sets the stage.
